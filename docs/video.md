# How to Make Your 5+ Minute FormCMS Demo Video Using CapCut

**Goal**: Create a clear tutorial video showing these 5 steps:
1. Go to Render, create an app using FormCMS Docker image  
2. After the web app is ready, go to the website  
3. Configure the web app (admin user password + API key)  
4. Use the AI “create schema” feature to create schema and page  
5. Demo how to view the page and input data  

## Step-by-Step Guide Using CapCut (Free Version)

### 1. Record Your Screen
- Go to **[www.capcut.com](https://www.capcut.com)** → Click **“Create”** → Choose **“Screen Recorder”** (or use the **Desktop App** – recommended).
- Record the entire process in **one continuous take** or in **5 separate short clips**.
- Tips:
  - Speak naturally while recording, or stay quiet and add voiceover later.
  - Move mouse slowly and pause briefly on important clicks.
  - Record at 1080p or higher if possible.

### 2. Start a New Project
- Open CapCut (web or desktop version).
- Click **“New Project”**.
- Import your recording(s) — they will appear automatically if you used CapCut’s recorder, or drag & drop your MP4 files.

### 3. Edit the Video (Make it Professional)

**A. Organize into 5 Clear Steps**
- Use the **scissors** tool to split the recording into 5 sections.
- Arrange them in the correct order on the timeline.
- Trim any unnecessary waiting time.

**B. Add Voiceover / Narration**
- **Option 1**: Record your own voice directly in CapCut (**Audio → Voiceover**).
- **Option 2 (Recommended)**: Use **AI Text-to-Speech**:
  - Type your script for each step.
  - Choose a natural-sounding voice.
  - Let CapCut generate and sync the audio automatically.

**C. Add Auto Captions**
- Select the video → Click **“Captions”** → **“Auto Captions”**.
- Free version supports up to **10 minutes** per video — enough for your 5–8 minute demo.
- Customize caption style (font, color, position, animation).

**D. Add Step Labels (Very Important)**
- For each section, add large text on screen:
  - “Step 1: Deploy on Render.com”
  - “Step 2: Open the Live Website”
  - “Step 3: Configure Admin & API Key”
  - etc.
- Use animated text templates for a clean, professional look.

**E. Add Zooms & Highlights**
- Use **Keyframe** or **Auto Zoom** effects to zoom in on buttons, fields, and important actions.
- Highlight mouse clicks when necessary.

**F. Final Polish**
- Add simple transitions between steps (fade or slide).
- Add soft background music (keep volume low).
- Adjust playback speed for any slow waiting parts.

### 4. Export the Video
- Click **Export** (top right).
- Resolution: **1080p** (free version).
- Format: MP4.
- **No watermark** on the main export.

### Estimated Time
- Recording: 10–15 minutes
- Editing: 30–60 minutes (first time)
